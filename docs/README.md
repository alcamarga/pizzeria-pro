# Docs — Pizzería Pro

Esta carpeta concentra la documentación operativa del proyecto (setup local, variables de entorno, API, despliegue y troubleshooting).

## Índice

- `setup.md`: instalación y ejecución local (backend y frontend)
- `env.md`: variables de entorno y configuración
- `api.md`: endpoints, autenticación y ejemplos de requests
- `deploy.md`: guía de despliegue (Render + Vercel)
- `security.md`: notas de seguridad (JWT, CORS)
- `troubleshooting.md`: errores comunes y soluciones rápidas
